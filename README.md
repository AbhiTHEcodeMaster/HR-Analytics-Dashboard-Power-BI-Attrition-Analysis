📊 HR Analytics Dashboard – Power BI Project
📌 Project Overview

This project presents an interactive HR Analytics Dashboard built using Microsoft Power BI. The dashboard analyzes employee data to identify key factors contributing to employee attrition.

It provides insights into workforce trends, salary distribution, job satisfaction levels, and department-wise attrition.

🎯 Business Objective

The objective of this project is to:

Analyze employee attrition patterns

Identify high-risk departments and job roles

Understand impact of overtime on attrition

Evaluate salary and satisfaction trends

Support HR decision-making with data insights

📊 Dashboard Features
🔹 Page 1 – HR Summary Dashboard

Total Employees

Total Attrition

Attrition Rate %

Average Salary

Attrition by Department

Attrition by Job Satisfaction

Attrition by Gender

Salary by Department

🔹 Page 2 – Attrition Deep Dive

Attrition by Job Role

Attrition by Years at Company

Attrition by Overtime

Salary vs Attrition Analysis

Interactive Filters (Department, Gender, Education Field)

Data Modeling

Data Visualization

HR Analytics Concepts

📈 Key Insights

Certain departments show higher attrition rates.

Overtime employees tend to have higher attrition.

Lower job satisfaction correlates with increased attrition.

Experience level impacts employee retention.
